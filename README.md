# Hotel Bookings Dataset Analysis

## 📊 Overview
Analyzing hotel reservation data to understand complex booking trends, identify primary drivers of cancellation rates, and forecast peak periods for better resource management.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Matplotlib

## 📈 Key Findings
- Identified key factors contributing to booking cancellations.
- Visualized lead-time patterns to assist in resource allocation for the hotel.
